description: Automatically push backend changes to GitHub after implementation
---
Whenever changes are made to the `quiz-backend` codebase:
1. Run `git status` in `c:\Users\ali akbar\Desktop\gk app\quiz-backend` to check for modified files.
2. Run `git add .` to stage the changes.
3. Run `git commit -m "Descriptive commit message"` with a summary of the changes.
4. Run `git push origin main` to deploy the changes to the remote repository.
5. Notify the user once the push is successful.
